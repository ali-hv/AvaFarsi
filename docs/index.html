Download & Install

You can install AvaFarsi on any operating system

  • Windows
  • Linux
  • Mac

Windows

Just download the setup file from below and install the program.

AvaFarsi-2.1.1-64x.exe

Then you can run the AvaFarsi from your apps menu or desktop shortcut.

Linux

First of all, make sure you have python3 and pip and git installed. Then open your terminal and enter the following commands in order:

  • git clone https://github.com/Ali-Hosseinverdi/AvaFarsi.git
  • cd AvaFarsi && pip3 install -r requirements.txt
  • chmod +x install && sudo ./install

Then you can run the AvaFarsi from your apps menu or you can just enter the app name in your terminal.

Mac

First of all, make sure you have python3 and pip and git installed. Then open your terminal and enter the following commands in order:

  • git clone https://github.com/Ali-Hosseinverdi/AvaFarsi.git
  • cd AvaFarsi && pip3 install -r requirements.txt
  • chmod +x install && sudo ./install

Then you can run the AvaFarsi from your apps menu or you can just enter the app name in your terminal.

AvaFarsi provides the following features:

  • Download song by giving the name of the singer
  • Download song by giving the name of the song
  • Download song by giving the text of a part of the song
  • Download all of the songs at once
  • Download a certain number of songs by giving a number
